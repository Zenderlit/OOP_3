Лабораторная Работа№3
Цель: Необходимо закодировать объект Person в строку, записать это строку в файл, а затем
воссоздать из файла.

В функциональном подходе добавлены функции для сохранения/загрузки из файлов, что демонстрирует практическое применение сериализации.
Все реализации используют общий паттерн:

1.При сериализации: рекурсивный обход графа с отслеживанием посещенных объектов
2.Сохранение ссылок как ID вместо самих объектов
3.При десериализации: создание объектов → восстановление связей по ID
